Dynamic Programming Problems - Comprehensive Guide

1. Linear DP (1D Problems)​

  1. Climbing Stairs - Calculate ways to reach the top with 1 or 2 steps at a time
  2. House Robber - Maximize amount without robbing adjacent houses
  3. Maximum Subarray (Kadane's) - Find contiguous subarray with maximum sum
  4. Coin Change - Minimum coins needed to make target amount
  5. Coin Change II - Count number of ways to make target amount
  6. Decode Ways - Count valid ways to decode a numeric string
  7. Min Cost Climbing Stairs - Minimum cost to reach top with variable step costs
  8. Jump Game - Determine if you can reach the last index
  9. Jump Game II - Minimum jumps to reach the last index
  10. Delete and Earn - Maximize points by deleting elements and their neighbors
  11. Fibonacci Number - Classic DP introduction problem
  12. N-th Tribonacci Number - Extension of Fibonacci
  13. Perfect Squares - Minimum perfect squares that sum to n
  14. Integer Break - Maximize product by breaking an integer
  15. Arithmetic Slices - Count arithmetic subarrays

2. String DP​

  1. Longest Common Subsequence (LCS) - Find longest sequence appearing in same order in two strings
  2. Edit Distance - Minimum operations to convert one string to another
  3. Longest Palindromic Substring - Find longest palindrome within a string
  4. Word Break - Check if string can be segmented into dictionary words
  5. Word Break II - Return all possible word break combinations
  6. Palindrome Partitioning II - Minimum cuts to partition string into palindromes
  7. Longest Common Substring - Find longest contiguous matching substring
  8. Longest Palindromic Subsequence - Longest subsequence that is a palindrome
  9. Distinct Subsequences - Count distinct subsequences of one string in another
  10. Regular Expression Matching - Implement regex matching with '.' and '*'
  11. Wildcard Matching - Implement wildcard pattern matching
  12. Interleaving String - Check if string is interleaving of two strings
  13. Minimum ASCII Delete Sum - Minimum ASCII sum to make two strings equal
  14. Shortest Common Supersequence - Find shortest string containing both strings

3. Knapsack Variants​

  1. 0-1 Knapsack Problem - Maximize value within weight limit
  2. Subset Sum Problem - Check if subset exists that sums to target
  3. Partition Equal Subset Sum - Determine if array can be partitioned into equal sum subsets
  4. Target Sum - Count ways to assign +/- signs to reach target
  5. Rod Cutting - Maximize profit by cutting rod into pieces
  6. Unbounded Knapsack - Knapsack with unlimited items
  7. Minimum Subset Sum Difference - Partition array to minimize difference
  8. Count of Subset Sum - Count subsets with given sum
  9. Ones and Zeroes - 2D knapsack with binary strings
  10. Last Stone Weight II - Minimize final stone weight

4. Sequence DP​

  1. Longest Increasing Subsequence (LIS) - Find longest strictly increasing subsequence
  2. Maximum Sum Increasing Subsequence - LIS variant maximizing sum
  3. Number of Longest Increasing Subsequence - Count all LIS
  4. Max Length of Pair Chain - Find longest chain of pairs
  5. Russian Doll Envelopes - 2D LIS variant
  6. Box Stacking - Stack boxes to maximize height
  7. Longest Arithmetic Subsequence - Find longest arithmetic progression
  8. Wiggle Subsequence - Longest alternating subsequence
  9. Build Array Where You Can Find Maximum Exactly K Comparisons - Complex sequence construction

5. Multi-Dimensional DP​

  1. Unique Paths - Count paths in grid from top-left to bottom-right
  2. Unique Paths II - Unique paths with obstacles
  3. Minimum Path Sum - Find path with minimum sum in grid
  4. Longest Increasing Path in Matrix - Find longest increasing path
  5. Maximal Square - Find largest square of 1s in binary matrix
  6. Maximal Rectangle - Find largest rectangle of 1s
  7. Triangle - Minimum path sum in triangle
  8. Dungeon Game - Minimum initial health to reach princess
  9. Cherry Pickup - Maximize cherries in grid with round trip
  10. Cherry Pickup II - Two robots collecting cherries
  11. Minimum Falling Path Sum - Minimum sum path falling through matrix
  12. Super Egg Drop - Minimize trials to find critical floor

6. Interval DP​

  1. Matrix Chain Multiplication - Minimize operations for matrix multiplication
  2. Burst Balloons - Maximize coins by bursting balloons optimally
  3. Boolean Parenthesization - Count ways to parenthesize boolean expression
  4. Palindrome Partitioning II - Minimum cuts to partition string into palindromes
  5. Minimum Cost to Merge Stones - Merge piles of stones optimally
  6. Remove Boxes - Maximize points by removing boxes
  7. Predict the Winner - Optimal game strategy
  8. Minimum Score Triangulation of Polygon - Triangulate polygon with minimum score

7. Tree DP​

  1. Unique Binary Search Trees - Count structurally unique BSTs with n nodes
  2. Unique Binary Search Trees II - Generate all unique BSTs
  3. House Robber III - Rob houses arranged in binary tree without adjacent nodes
  4. Binary Tree Maximum Path Sum - Find path with maximum sum
  5. Diameter of Binary Tree - Find longest path between any two nodes
  6. Longest Univalue Path - Longest path with same values
  7. Maximum Product of Splitted Binary Tree - Split tree to maximize product

8. Stock Trading Problems​

  1. Best Time to Buy and Sell Stock - Maximize profit with single transaction
  2. Best Time to Buy and Sell Stock II - Unlimited transactions allowed
  3. Best Time to Buy and Sell Stock III - At most two transactions
  4. Best Time to Buy and Sell Stock IV - At most K transactions
  5. Best Time to Buy and Sell Stock with Cooldown - One day cooldown after selling
  6. Best Time to Buy and Sell Stock with Transaction Fee - Pay fee on each transaction

9. Game Theory DP​

  1. Optimal Strategy for a Game - Maximize coins picking from ends of array
  2. Predict the Winner - Determine if player can win two-player game
  3. Stone Game - Optimal play with piles of stones
  4. Stone Game II - Extended stone game with variable picks
  5. Stone Game III - Pick 1-3 stones optimally
  6. Stone Game IV - Remove square numbers of stones
  7. Stone Game V - Divide and maximize score
  8. Nim Game - Classic game theory problem
  9. Cat and Mouse - Graph game with optimal strategies

10. Probability DP​

  1. Knight Probability in Chessboard - Probability knight remains on board
  2. Soup Servings - Probability of soup distribution outcomes
  3. New 21 Game - Probability of winning card game
  4. Dice Roll Simulation - Count valid dice roll sequences
  5. Number of Dice Rolls With Target Sum - Count ways to achieve target sum

11. Bitmask DP​

  1. Traveling Salesman Problem - Classic TSP with bitmask
  2. Maximum Students Taking Exam - Seat students optimally
  3. Shortest Path Visiting All Nodes - Visit all nodes with minimum path
  4. Partition to K Equal Sum Subsets - Partition array into k subsets
  5. Fair Distribution of Cookies - Minimize unfairness in distribution
  6. Find Minimum Time to Finish All Jobs - Distribute jobs among workers
  7. Number of Ways to Wear Different Hats - Assign hats to people

12. Digit DP​

  1. Count Numbers with Unique Digits - Count n-digit numbers with unique digits
  2. Numbers At Most N Given Digit Set - Count valid numbers
  3. Non-negative Integers without Consecutive Ones - Count binary numbers
  4. Numbers With Repeated Digits - Count numbers with at least one repeated digit

Practice Strategy​

Beginner Path​

Start with Linear DP → String DP (basic problems) → Knapsack variants

Intermediate Path​

Sequence DP → Multi-Dimensional DP → Stock Trading → Tree DP

Advanced Path​

Interval DP → Game Theory → Bitmask DP → Digit DP → Probability DP

Tips​

  • Master state definition and transition equations
  • Draw recursion trees for intuition
  • Practice space optimization techniques
  • Learn both top-down (memoization) and bottom-up (tabulation) approaches
  • Recognize problem patterns to apply correct DP technique
