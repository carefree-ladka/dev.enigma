2. Depth First Search (DFS)​

Basic DFS​

  1. Number of Islands - Classic DFS on grid
  2. Max Area of Island - Find largest connected component
  3. Flood Fill - Color connected region
  4. Number of Provinces - Count connected components
  5. Find All Groups of Farmland - Identify rectangular regions
  6. Keys and Rooms - Reachability problem

DFS with Backtracking​

  1. All Paths From Source to Target - Find all paths
  2. Path With Maximum Gold - Find optimal path with backtracking
  3. Word Search - Search word in grid
  4. Word Search II - Multiple words search (DFS + Trie)
  5. Combination Sum - DFS with combinations
  6. Palindrome Partitioning - Partition with backtracking

DFS for Cycle Detection​

  1. Course Schedule - Detect cycle in directed graph
  2. Course Schedule II - Topological sort with DFS
  3. Graph Valid Tree - Check if graph is tree
  4. Redundant Connection - Find edge creating cycle

Advanced DFS​

  1. Number of Enclaves - Count unreachable cells
  2. Surrounded Regions - Capture regions
  3. Number of Closed Islands - Islands not touching boundary
  4. Count Sub Islands - Islands subset problem
  5. Pacific Atlantic Water Flow - Reverse DFS from boundaries
  6. Longest Increasing Path in a Matrix - DFS with memoization

3. Breadth First Search (BFS)​

Basic BFS​

  1. Binary Tree Level Order Traversal - Level by level traversal
  2. Minimum Depth of Binary Tree - Shortest path to leaf
  3. Shortest Path in Binary Matrix - 8-directional shortest path
  4. Shortest Bridge - Connect two islands
  5. Nearest Exit from Entrance in Maze - Find nearest exit
  6. Jump Game III - Reach zero using BFS

BFS for Shortest Path​

  1. 01 Matrix - Distance to nearest 0
  2. As Far from Land as Possible - Furthest from land
  3. Walls and Gates - Distance to gate
  4. Rotting Oranges - Multi-source BFS
  5. Shortest Path to Get Food - Grid shortest path
  6. Snakes and Ladders - Board game shortest path

BFS State Space Search​

  1. Open the Lock - State space BFS
  2. Minimum Genetic Mutation - String transformation
  3. Word Ladder - Word transformation chain
  4. Word Ladder II - All shortest transformation sequences
  5. Sliding Puzzle - Puzzle solving with BFS
  6. Minimum Knight Moves - Chess knight movement

Advanced BFS​

  1. Cut Off Trees for Golf Event - Multiple BFS searches
  2. Minimum Moves to Reach Target with Rotations - Complex state BFS
  3. Bus Routes - Graph of routes
  4. Shortest Path Visiting All Nodes - BFS with bitmask

Basic Union Find​

  1. Number of Provinces - Connected components
  2. Number of Connected Components in an Undirected Graph - Count components
  3. Graph Valid Tree - Validate tree using union find
  4. Redundant Connection - Find redundant edge
  5. Redundant Connection II - Directed graph variant
  6. Most Stones Removed with Same Row or Column - Union by row/column

Union Find with Ranking​

  1. Accounts Merge - Merge by common emails
  2. Smallest String With Swaps - Character swapping
  3. Satisfiability of Equality Equations - Equation validation
  4. Evaluate Division - Union find with weights
  5. Sentence Similarity II - Transitive similarity

Advanced Union Find​

  1. Number of Islands II - Dynamic connectivity
  2. Swim in Rising Water - Binary search + union find
  3. Minimize Malware Spread - Component analysis
  4. Minimize Malware Spread II - Advanced variant
  5. Checking Existence of Edge Length Limited Paths - Offline queries
  6. Number of Good Paths - Union find with sorting
  7. Largest Component Size by Common Factor - Prime factorization + UF

  1. Course Schedule - Detect cycle
  2. Course Schedule II - Return topological order
  3. Minimum Height Trees - Find tree centers
  4. Alien Dictionary - Derive character order
  5. Sequence Reconstruction - Validate unique order
  6. Sort Items by Groups Respecting Dependencies - Nested topological sort

DFS-based Topological Sort​

  1. Course Schedule IV - Prerequisite queries
  2. Parallel Courses - Minimum semesters
  3. Parallel Courses II - Limited courses per semester
  4. Find All Possible Recipes from Given Supplies - Recipe dependencies

Applications​

  1. Longest Path in a Directed Acyclic Graph - DP on DAG
  2. All Ancestors of a Node in a Directed Acyclic Graph - Find ancestors
  3. Build a Matrix With Conditions - 2D topological constraints

  3. Cheapest Flights Within K Stops - Shortest path with constraints
  4. Path With Minimum Effort - Minimize maximum effort
  5. Swim in Rising Water - Minimize maximum elevation
  6. Minimum Cost to Make at Least One Valid Path in a Grid - Grid modification cost

Modified Dijkstra​

  1. The Maze II - Shortest distance with rolling ball
  2. Reachable Nodes In Subdivided Graph - Modified edges
  3. Find the Shortest Superstring - TSP variant
  4. Minimum Weighted Subgraph With the Required Paths - Multiple sources/destinations

Advanced Applications​

  1. Design Graph With Shortest Path Calculator - Design with Dijkstra
  2. Find the City With the Smallest Number of Neighbors at a Threshold Distance - Distance threshold
  3. Shortest Path with Alternating Colors - Color constraints

  1. Network Delay Time - Can use Bellman Ford
  2. Cheapest Flights Within K Stops - Limited iterations
