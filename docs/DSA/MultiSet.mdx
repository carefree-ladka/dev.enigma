# Multiset in JavaScript

A **Multiset** (also known as a **bag**) is a data structure similar to a set, but it allows duplicate elements. In a multiset, each element can appear multiple times, and you can efficiently track the number of occurrences of each element.

This implementation of a multiset is built using a JavaScript `Map`, where each element is stored as a key, and the number of times it appears is stored as the corresponding value.


```JavaScript
class Multiset {
  constructor() {
    this.set = new Map();
  }

  // Adds an element to the multiset
  add(element) {
    this.set.set(element, (this.set.get(element) ?? 0) + 1);
  }

  // Removes one occurrence of the element from the multiset
  remove(element) {
    if (this.set.has(element)) {
      const count = this.set.get(element);
      if (count > 1) {
        this.set.set(element, count - 1);
      } else {
        this.set.delete(element);
      }
    }
  }

  // Returns the count of occurrences of the element
  count(element) {
    return this.set.get(element) ?? 0;
  }

  // Checks if an element exists in the multiset
  has(element) {
    return this.set.has(element);
  }

  // Returns the size of the multiset (unique elements count)
  size() {
    return this.set.size;
  }

  // Removes all elements from the multiset
  clear() {
    this.set.clear();
  }

  // Returns the entire multiset as a plain object (for inspection)
  toObject() {
    return Object.fromEntries(this.set);
  }
}

const multiset = new Multiset();

// Add elements to the multiset
multiset.add('apple');
multiset.add('banana');
multiset.add('apple');

// Check the count of occurrences
console.log(multiset.count('apple'));  // 2
console.log(multiset.count('banana')); // 1

// Remove one occurrence of 'apple'
multiset.remove('apple');
console.log(multiset.count('apple'));  // 1

// Check if an element exists
console.log(multiset.has('apple'));    // true
console.log(multiset.has('orange'));  // false

// Get the size (number of unique elements)
console.log(multiset.size()); // 2 (apple and banana)

// Get the entire multiset
console.log(multiset.toObject()); // { apple: 1, banana: 1 }

// Clear the multiset
multiset.clear();
console.log(multiset.toObject()); // {}
```
