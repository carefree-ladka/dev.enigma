10. Advanced Heap Problems​

Matrix Problems​

  1. Kth Smallest Element in a Sorted Matrix - Binary search or heap
  2. Find K-th Smallest Pair Distance - Binary search + counting
  3. Kth Smallest Prime Fraction - Matrix traversal with heap
  4. Find the K-th Smallest Sum of a Matrix With Sorted Rows - K-way merge variant

Complex Heap Maintenance​

  1. Sliding Window Maximum - Deque or multiset (heap alternative)
  2. Last Stone Weight II - DP problem (not typical heap)
  3. Furthest Building You Can Reach - Greedy with heap
  4. Minimum Cost to Connect Sticks - Huffman encoding pattern
  5. Minimize Deviation in Array - Max heap with halving

Optimization Problems​

  1. Maximum Performance of a Team - Greedy with min heap
  2. Minimum Number of Refueling Stops - Max heap for fuel stations
  3. Maximum Subsequence Score - Sort + min heap

  5. Longest Happy String - Greedy with max heap
  6. Construct Target Array With Multiple Sums - Reverse engineering with heap
