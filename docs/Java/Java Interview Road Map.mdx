# Java Interview Preparation Roadmap

---

## 1. Core Java

| Topic | Subtopics | Example Questions / Practice |
|-------|-----------|-----------------------------|
| **OOP Concepts** | Inheritance, Polymorphism (compile/runtime), Abstraction, Encapsulation, Interfaces | Explain runtime vs compile-time polymorphism.<br /> Design a shape class hierarchy with method overriding. |
| **Java Basics** | `main` method, packages, access modifiers | - Difference between `public`, `private`, `protected`, default.<br />- Explain `static` keyword and its use cases. |
| **Data Types & Variables** | Primitive vs Wrapper, autoboxing/unboxing | - Difference between `int` and `Integer`.<br />- What happens during autoboxing? |
| **Strings** | `String`, `StringBuilder`, `StringBuffer`, immutability | - Why `String` is immutable?<br />- Compare `StringBuilder` vs `StringBuffer`. |
| **Collections Framework** | List, Set, Map, Queue, implementations, differences | - Difference between `ArrayList` and `LinkedList`.<br />- Implement LRU cache using `LinkedHashMap`. |
| **Generics** | Bounded types, wildcards, type erasure | - Difference between `? extends T` and `? super T`.<br />- Why Java uses type erasure? |
| **Exception Handling** | Checked vs unchecked, `throw` vs `throws`, custom exceptions | - Write a custom exception.<br />- Explain try-with-resources. |
| **Java Memory Model** | Stack vs Heap, GC, memory leaks, references | - Explain strong, weak, soft, phantom references.<br />- What is a memory leak in Java? |
