DOM Cheatsheet

Finding HTML Elements​

document.getElementById(id) Returns a single element with the specified ID.

document.getElementsByClassName(className) Returns a live HTMLCollection of elements with the specified class name.

document.getElementsByTagName(tagName) Returns a live HTMLCollection of elements with the specified tag name.

document.querySelector(cssSelector) Returns the first element that matches the specified CSS selector.

document.querySelectorAll(cssSelector) Returns a static NodeList of all elements that match the specified CSS selector.

Traversing the DOM​

parentNode/parentElement Gets the parent node/element of the current element.

childNodes/children Gets a list of all child nodes/elements of the current element.

firstChild/firstElementChild Gets the first child node/element of the current element.

lastChild/lastElementChild Gets the last child node/element of the current element.

nextSibling/nextElementSibling Gets the next sibling node/element of the current element.

previousSibling/previousElementSibling Gets the previous sibling node/element of the current element.

Adding Elements​

createElement(tagName) Creates a new element with the specified tag name.

createTextNode(text) Creates a new text node with the specified text content.

appendChild(node) Adds a node as the last child of a parent element.

insertBefore(newNode, referenceNode) Inserts a new node before another child element in the parent node.

insertAdjacentHTML(position, html) Inserts the specified HTML text into the DOM tree at the specified position.

  • beforebegin: Before the element.
  • afterbegin: Inside the element, before its first child.
  • beforeend: Inside the element, after its last child.
  • afterend: After the element.

insertAdjacentElement(position, element) Inserts the specified element node at the specified position.

insertAdjacentText(position, text) Inserts the specified text node at the specified position.

Removing Elements​

removeChild(node) Removes a child node from the DOM.

replaceChild(newNode, oldNode) Replaces an existing child node with a new node.

classList Provides methods to add, remove, toggle, and check the existence of classes on an element.

setAttribute(name, value) Sets the value of the specified attribute on the current element.

getAttribute(name) Returns the value of the specified attribute on the current element.

Event Handling​

addEventListener(event, function, options) Attaches an event listener to the current element.

removeEventListener(event, function, options) Removes an event listener from the current element.

Some common events:

  • click
  • keydown
  • keyup
  • mouseover
  • mouseout
  • submit
  • load
  • resize

Additional DOM Properties and Methods​

textContent Gets or sets the text content of the current element and its descendants.

innerHTML Gets or sets the HTML content of the current element.

outerHTML Gets or sets the HTML content of the current element, including the element itself.

style Provides access to the inline styles of the current element.

scrollIntoView() Scrolls the current element into the visible area of the browser window.

getBoundingClientRect() Returns the size of an element and its position relative to the viewport.

dispatchEvent(event) Dispatches an event to the current element.

focus() Sets the focus on the current element.

blur() Removes the focus from the current element.

Remember, the DOM API provides a wide range of methods and properties for interacting with and manipulating the HTML document. The cheatsheet above covers some of the most commonly used ones, but there are many more available depending on your specific needs.
